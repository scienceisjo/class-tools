🔧 선생님 전용
🚶 돌아다니기 — 바닥을 터치하면 그곳으로 걸어가요. 드래그=시점 회전(상하=높낮이), 두 손가락/마우스 휠=확대·축소. 방향키·WASD, 미니맵 터치도 OK.
🎂 생일 게시판 — 생일 광장의 게시판을 클릭(또는 메뉴 → 우리 반 생일 게시판)하면 우리 반 전체 생일을 월별로 볼 수 있어요.
👥 함께 있기 — 같은 방(반) 코드로 접속한 친구들이 같은 숲에 실시간으로 보여요. 위로 걸어가면 캐릭터 뒷모습(꼬리)도 보인답니다!
👀 관람하기 — 마음 근처에 가면 말풍선 미리보기가 떠요. 말풍선(또는 Space)을 누르면 크게 보기 + ❤️ 공감.
📖 모아보기 — 어떤 공간에 도착하면 아래에 '이 곳 글 모아보기' 버튼이 떠요(팻말을 눌러도 됨). 위쪽 카테고리(칩)를 누르면 그 종류의 글만 모아 볼 수 있어요.
🗺️ 일곱 공간 — 🌳다짐나무(다짐·우리 반 모습) / ⭐별빛갤러리(순간·추억 사진) / 🎪추억광장(행사) / 📮마음우체국(친구에게 한마디) / 🎂생일광장(다가오는 생일 게시판·축하 한마디) / 🤝1인1역 본부(점검·스타 추천·🌟숨은 기여 자랑·만들고 싶은 역할) / 🌱도전온실(방학 중 도전)
🏆 1인1역 스타 — 추천 글에 ❤️ 공감을 누르는 게 곧 투표! 발표 모드에서 공감 수를 함께 보여줘요.
🛠️ 교사용 — 메뉴 → 🔒 교사 모드에 비밀번호를 넣으면 관리 메뉴가 열려요. 비번은 서버에서 확인하므로 소스코드를 봐도 못 뚫어요. 학생은 읽기·마음 남기기만 가능(공지·명단·삭제 불가). 전자칠판에서 켰다면 넘기기 전 교사 모드 끄기! (처음 한 번 파일 하단 SECURITY SQL 실행 필요)
🎥 전자칠판 — 메뉴 → 자동 관람 모드를 켜면 카메라가 숲을 천천히 돌며 친구들의 활동을 비춰줘요. 터치하면 해제.
