рҹ‘Ҙ н•Ёк»ҳ мһҲкё° вҖ” к°ҷмқҖ л°©(л°ҳ) мҪ”л“ңлЎң м ‘мҶҚн•ң м№ңкө¬л“Өмқҙ к°ҷмқҖ мҲІм—җ мӢӨмӢңк°„мңјлЎң ліҙм—¬мҡ”. мң„лЎң кұём–ҙк°Җл©ҙ мәҗлҰӯн„° л’·лӘЁмҠө(кј¬лҰ¬)лҸ„ ліҙмқёлӢөлӢҲлӢӨ!
рҹ‘Җ кҙҖлһҢн•ҳкё° вҖ” кіөк°„(кІҢмӢңнҢҗ)м—җ к°Җкұ°лӮҳ нҢ»л§җВ·л§ҲмқҢмқ„ лҲ„лҘҙл©ҙ к·ёкіі кёҖмқҙ н•ң лІҲм—җ лӘЁм—¬мҡ”. кІҢмӢңнҢҗм—җм„ң мҳ¬лҰ° мӮ¬лһҢВ·лӮ м§ңлЎң кіЁлқј мқҪкі  вқӨпёҸ кіөк°җ.
рҹ“– лӘЁм•„ліҙкё° вҖ” м–ҙл–Ө кіөк°„м—җ лҸ„м°©н•ҳл©ҙ м•„лһҳм—җ 'мқҙ кіі кёҖ лӘЁм•„ліҙкё°' лІ„нҠјмқҙ л– мҡ”(нҢ»л§җмқ„ лҲҢлҹ¬лҸ„ лҗЁ). мң„мӘҪ м№ҙн…Ңкі лҰ¬(м№©)лҘј лҲ„лҘҙл©ҙ к·ё мў…лҘҳмқҳ кёҖл§Ң лӘЁм•„ ліј мҲҳ мһҲм–ҙмҡ”.
рҹ—әпёҸ мқјкіұ кіөк°„ вҖ” рҹҢілӢӨм§җлӮҳл¬ҙ(лӢӨм§җВ·мҡ°лҰ¬ л°ҳ лӘЁмҠө) / вӯҗлі„л№ӣк°Өлҹ¬лҰ¬(мҲңк°„В·м¶”м–ө мӮ¬м§„) / рҹҺӘм¶”м–өкҙ‘мһҘ(н–үмӮ¬) / рҹ“®л§ҲмқҢмҡ°мІҙкөӯ(м№ңкө¬м—җкІҢ н•ңл§Ҳл””) / рҹҺӮмғқмқјкҙ‘мһҘ(лӢӨк°ҖмҳӨлҠ” мғқмқј кІҢмӢңнҢҗВ·м¶•н•ҳ н•ңл§Ҳл””) / рҹӨқ1мқё1м—ӯ мҠӨн…Ңмқҙм…ҳ(м җкІҖВ·мҠӨнғҖ м¶”мІңВ·рҹҢҹмҲЁмқҖ кё°м—¬ мһҗлһ‘В·л§Ңл“Өкі  мӢ¶мқҖ м—ӯн• ) / рҹҢұлҸ„м „мҳЁмӢӨ(л°©н•ҷ мӨ‘ лҸ„м „)
рҹҸҶ 1мқё1м—ӯ мҠӨнғҖ вҖ” м¶”мІң кёҖм—җ вқӨпёҸ кіөк°җмқ„ лҲ„лҘҙлҠ” кІҢ кі§ нҲ¬н‘ң! л°ңн‘ң лӘЁл“ңм—җм„ң кіөк°җ мҲҳлҘј н•Ёк»ҳ ліҙм—¬мӨҳмҡ”.
рҹӣ пёҸ көҗмӮ¬мҡ© вҖ” л©”лүҙ вҶ’ рҹ”’ көҗмӮ¬ лӘЁл“ңм—җ л№„л°ҖлІҲнҳёлҘј л„Јмңјл©ҙ кҙҖлҰ¬ л©”лүҙк°Җ м—ҙл Өмҡ”. л№„лІҲмқҖ м„ңлІ„м—җм„ң нҷ•мқён•ҳлҜҖлЎң мҶҢмҠӨмҪ”л“ңлҘј лҙҗлҸ„ лӘ» лҡ«м–ҙмҡ”. н•ҷмғқмқҖ мқҪкё°В·л§ҲмқҢ лӮЁкё°кё°л§Ң к°ҖлҠҘ(кіөм§ҖВ·лӘ…лӢЁВ·мӮӯм ң л¶Ҳк°Җ). м „мһҗм№ нҢҗм—җм„ң мј°лӢӨл©ҙ л„ҳкё°кё° м „ көҗмӮ¬ лӘЁл“ң лҒ„кё°!
